LoRA Training Experiment

A little experiment project I am doing to train a LoRA model to mimic my sketchy art style. Utilising open-sourced software and resources including Stable Diffusion, ComfyUI, HuggingFace, Google Collab.

More Updates are coming soon!

0: Setting up and preparing

02/07/2025

  • Taking pictures of the sketches and upload them to my computer.
  • Process each image to remove the background and resize them to 720x720 (should have done 768x768).
  • Using ComfyUI and WD Tagger to automate tagging all my sketches.
  • Made a custom Python script to gather images and the generated tags into a single Excel and JSON file for easier edits.
  • Within the Excel, add in my manuel tags and description for each image.
  • Use Python to export the data (manuel and generated tags) back out in to .txt files.
  • Following this guide: https://civitai.com/models/22530/guide-make-your-own-loras-easy-and-free
  • Upload my sketches and tags (36 in total) in to Google Drive and use there Google Collab project to train the LoRA
  • Output was not close to my style, but it did detect that it converts images to pen sketches.
  • TODO: Sketch more to grow the dataset, use other tagging models like BLIP.
